# نظام الامتحانات التفاعلي

نظام امتحانات متكامل للمعلمين والطلاب - Next.js + Supabase + Vercel

## المميزات
- لوحة تحكم لإنشاء وإدارة الامتحانات
- دعم كامل للغة العربية (خط Rubik)
- خلط الإجابات تلقائياً لمنع الغش
- توقيت الامتحان (اختياري)
- ترتيب الطلاب بحسب الدرجة (Real-time)
- مشاركة رابط الامتحان
- يعمل على الموبايل والكمبيوتر
- شاشة نتيجة تفاعلية مع تهنئة للناجح

## خطوات الإعداد

### 1. إعداد Supabase
1. أنشئ مشروعاً على supabase.com
2. افتح SQL Editor وشغّل ملف supabase-schema.sql
3. انسخ Project URL و anon key

### 2. متغيرات البيئة
```
cp .env.local.example .env.local
```
ثم أضف قيمك في .env.local

### 3. تشغيل محلي
```
npm install
npm run dev
```

## النشر على Vercel
1. ارفع على GitHub
2. اربط بـ Vercel
3. أضف متغيرات البيئة
4. Deploy!

## روابط الصفحات
- / → لوحة التحكم
- /admin/new → إنشاء امتحان
- /admin/edit/[id] → تعديل امتحان
- /admin/results/[id] → نتائج وترتيب
- /exam/[id] → صفحة الطالب
